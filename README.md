# 🎥 YouTube RAG ChatBot

YouTube RAG ChatBot is an AI-powered application built with **Python, Streamlit, LangChain, Hugging Face, FAISS, and the YouTube Transcript API**. The application allows users to paste a YouTube video URL, automatically extracts the video's transcript and metadata, splits the transcript into meaningful chunks, generates vector embeddings, and stores them in a FAISS vector database for efficient semantic search. When a user asks a question about the video, the chatbot retrieves the most relevant transcript chunks and uses the **Qwen2.5-7B-Instruct** language model to generate accurate, context-aware answers. The project demonstrates the practical implementation of **Retrieval-Augmented Generation (RAG)**, semantic search, vector databases, and large language models while providing a clean, interactive Streamlit interface for an intuitive user experience.
